# DocuFlow AI — Screenshots Guide

This guide lists the screenshots recommended to include in your repository's `/docs/images` folder to showcase the platform's UI capabilities.

---

## Recommended Screenshots List

### 1. `01_dashboard.png`
* **Target Layout**: The main dashboard page (`/dashboard`).
* **Visual Highlights**: Shows the active KPI summary cards (Total Spent, Exception Rate, Processing Time), exception distribution charts, and the recent documents list.

### 2. `02_login.png`
* **Target Layout**: The login page (`/login`).
* **Visual Highlights**: Minimalist secure login form showing email, password inputs, and role indicators.

### 3. `03_upload.png`
* **Target Layout**: Invoice upload page (`/documents/upload`).
* **Visual Highlights**: Drag-and-drop file uploader zone with file selection states.

### 4. `04_document_detail.png`
* **Target Layout**: Document details view (`/documents/$id`).
* **Visual Highlights**: Displays the metadata cards (PO Number, Amount, Confidence score) and status badges.

### 5. `05_ocr_raw_text.png`
* **Target Layout**: Document detail page with "Raw OCR Text" tab active.
* **Visual Highlights**: Shows the extracted raw text next to the digital preview panel.

### 6. `06_validation_results.png`
* **Target Layout**: Validation Results section on the document detail page.
* **Visual Highlights**: Displays the validation checklist showing Passed/Failed/Warning rules with expected vs actual values.

### 7. `07_inline_corrections.png`
* **Target Layout**: Extracted Fields table during editing.
* **Visual Highlights**: Shows inline edit input box and action buttons when a Reviewer updates a value.

### 8. `08_reviewer_notes.png`
* **Target Layout**: Reviewer Notes alert box and the Complete Review dialog.
* **Visual Highlights**: Displays comments entered by the reviewer during human verification.

### 9. `09_audit_timeline.png`
* **Target Layout**: Audit timeline section at the bottom of the page.
* **Visual Highlights**: Displays chronological events list showing actions, actors, values, and timestamps.
