# Security Policy

We take the security of DocuFlow AI seriously. If you believe you have found a security vulnerability, please follow the guidelines below to report it.

## Supported Versions
Only version **1.0.x** of DocuFlow AI is currently supported with security updates.

## Reporting a Vulnerability
Please do not report security vulnerabilities through public GitHub issues. Instead, email reports to [security@docuflow.ai](mailto:security@docuflow.ai) with the following information:
* Description of the vulnerability and its potential impact.
* Steps to reproduce the issue (including proof-of-concept scripts or payloads).
* Any details regarding your test environment.

We will acknowledge receipt of your report within 48 hours and work with you to coordinate a responsible disclosure timeline.
